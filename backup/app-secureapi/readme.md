# app-secureapi

## features

- ApplicationRunner
- AuthenticationFilter
- AuthenticationManager
- AuthEntryPoint
- JSON Web Token
- OncePerRequestFilter
- PasswordEncoder
- SecurityFilterChain
- Spring Boot Security
- UserDetails

## commands

```bash
mvn clean compile
mvn -Dtests=Unit test
mvn -Dtests=Integration test
```
