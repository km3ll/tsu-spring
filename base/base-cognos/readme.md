# base-cognos

## features

- Get Mapping
- Google Java Format
- Maven Modules
- Maven Surefire Plugin
- Maven Spotless Plugin
- Request Mapping
- Rest Controller
- Spring Boot Test
- Spring Boot Web

## commands

```bash
mvn clean compile
mvn clean install
mvn clean test

mvn spotless:check
mvn spotless:apply

mvn -pl api spring-boot:run
```

## structure

```
cognos
├── api
```
